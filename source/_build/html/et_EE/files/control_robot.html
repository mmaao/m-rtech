Roboti juhtimine¶

ROBOTONDI juhtimine¶

Seadistamine¶

  1. Paigalda teleop twist keyboard

    sudo apt update
sudo apt install ros-melodic-teleop-twist-keyboard

  2. Ava enda Androidi seadmes Google Play pood ning paigalda ROS Control app.

  3. Loo roboti ja arvuti vahel ssh ühendus, nagu näidatud siin: Kaugteel robotisse sisenemine.

  4. Käivita draiver

    roslaunch robotont_driver driver_basic.launch

Roboti juhtimine teleop twist keyboardi abil¶

  1. Ava uus terminali aken

  2. Sisesta järgnev käsk:

    rosrun teleop_twist_keyboard teleop_twist_keyboard.py __ns:=/robotont

  3. Kasuta roboti liigutamiseks järgnevaid klahve:

    Hoiatus

    Sellest hetkest saab klahvivajutustega panna robotit eri suundades sõitma. Kui sa peaksid kaotama roboti üle kontrolli, tee üht järgnevatest:

    • ROBOTI PEATAMISEKS VAJUTA „k“ KLAHVI!
    • VAJUTA ROBOTIL OLEVAT PUNAST E-STOP NUPPU.

    Vihje

    Pane tähele, et klahvivajutused mõjuvad ainult siis kui terminaliaken on aktiivne.

    Nõuanne

    Sõlme peatamiseks kasuta CTRL + C.

Roboti juhtimine Android seadmega¶

  1. Ava oma telefonis ROS Control rakendus

  2. Insert the ROBOTONT’s IP address into Master URI field by entering the following:

    http://robotont_IP_address:11311

  3. Vajuta avanenud aknal „Show advanced options“ ja täida „Joystick“ ja „Odometry“ väljad järgnevalt: „robotont/cmd_vel“ ja „robotont/odom“

  4. Click OK to add the robot

  5. Nüüd saad listist roboti valida ja seda liigutada, kasutades juhtkangi nuppu.
