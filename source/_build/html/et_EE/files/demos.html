Demod¶

Enne alustamis loo roboti ja arvuti vahel ssh ühendus, nagu näidatud siin: Kaugteel robotisse sisenemine.

Demode jooksutamiseks on vaja järgnevaid pakette:

  1. 2D kaardistamise jaoks:

    sudo apt update
sudo apt install ros-melodic-depthimage-to-laserscan
sudo apt install ros-melodic-cartographer-ros
sudo apt install ros-melodic-move-base

  2. 3D kaardistamise jaoks:

    sudo apt install ros-melodic-rtabmap-ros

  3. AR jälgimine

    sudo apt install ros-melodic-ar-track-alvar

2D kaardistamine¶

  1. **ROBOTONDI pardaarvutis launch 2d_nav_carto.launch*

    roslaunch robotont_demos 2d_nav_carto.launch

  2. Kasutaja arvutis launch display_2d_mapping.launch, et visualiseerida tulemus.

    roslaunch robotont_demos display_2d_mapping.launch

  3. Roboti liigutamiseks ava kasutaja arvutis uus terminaliaken ja jooksuta teleop twist keyboard

    rosrun teleop_twist_keyboard teleop_twist_keyboard.py __ns:=/robotont

    Vihje

    Pane tähele, et klahvivajutused mõjuvad ainult siis kui terminaliaken on aktiivne.

  3. Roboti liigutamiseks ava kasutaja arvutis uus terminaliaken ja jooksuta teleop twist keyboard

AR jälgimine¶

  1. ROBOTONDI pardaarvutis launch ar_follow_the_leader.launch

    roslaunch roslaunch robotont_demos ar_follow_the_leader.launch

  2. Kasutaja arvutis launch display_ar_marker.launch, et visualiseerida tulemus

    roslaunch robotont_demos display_ar_marker.launch marker_id:=tag_nr

  3. Roboti liigutamiseks ava kasutaja arvutis uus terminaliaken ja jooksuta teleop twist keyboard

    rosrun teleop_twist_keyboard teleop_twist_keyboard.py __ns:=/robotont
